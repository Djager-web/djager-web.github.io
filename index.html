Twierdzenie Pitagorasa

a² + b² = c²
3² + 4² = c²
9 + 16 = c²
25 = c²
c = √25
c = 5
Sprawdzenie: 3² + 4² = 5² → 9 + 16 = 25 ✔

✏️ Zadania domowe

  • X + ... = ... – rozwiąż 3 przykłady samodzielnie.
  • Y – ... = ... – znajdź Y w trzech nowych przykładach.
  • A ÷ ... = ... – podstaw, policz i sprawdź.
  • Pitagoras – oblicz nieznany bok w trzech trójkątach.
